# SVG Illustration Skill

Expert at creating custom SVG illustrations, icons, mascots, and UI graphics.

## Features

- **Consultation-first**: Always asks user preferences before creating
- **Multiple styles**: Kawaii, minimal, hand-drawn, geometric
- **Ready-to-use output**: Finished SVGs saved directly to your project

## Trigger Phrases

- "create SVG"
- "icon for"  
- "mascot"
- "illustration"
- "cute graphic"
- "UI icon"
- `/svg-illustration`

## What It Creates

- App mascots (tomatoes, characters, animals)
- Profile avatars and badges
- UI icons and buttons
- Decorative elements
- Status indicators

## Workflow

1. **Consultation** - Asks what you need and your style preference
2. **Style Selection** - Confirms aesthetic direction
3. **Creation** - Generates optimized SVG code
4. **Delivery** - Saves to your specified location
5. **Iteration** - Refines based on feedback
